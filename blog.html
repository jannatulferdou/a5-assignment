Question-1: What are the different ways to select an element in the DOM?

There are some ways to select an element in the DOM. For example: getElementsByTagName(), getElementsByClassName(), getElementById(), querySelector(), querySelectorAll() .

Question-2: What is the difference between innerHTML, innerText, and textContent ?

"innerHTML" gives us the raw HTML, "innerText" only returns the visible text content of an element and "textContent" returns all text content within an element.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a JavaScript technique where we attach a single event listener to a parent element allow to handle the events.
